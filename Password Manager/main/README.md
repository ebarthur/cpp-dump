## Introduction

This C++ password manager is a basic console application that allows users to manage credentials. It supports adding, finding, and displaying credentials, as well as saving and loading them to/from a file.

**Note:** This is a basic example and does not include advanced security features. For a production-level application, encryption and additional security measures are essential.

## Features

- Add new credentials with a username and password.
- Find and display credentials based on the username.
- Display a list of all saved credentials.
- Save credentials to a file.
- Load credentials from a file.

## Usage

The program provides a simple menu with options to perform various actions:

1. **Add Credential:** Add a new credential with a username and password.
2. **Find Credential:** Find and display a credential based on the username.
3. **Display Credentials:** Display a list of all saved credentials.
4. **Save to File:** Save the current list of credentials to a file.
5. **Exit:** Save and exit the program.
